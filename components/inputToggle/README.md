# InputToggle

The component `InputToggle` creates a toggle.

## Demo

| Example | Description |
| :--- | :--- |
| <input-toggle></input-button> | Default toggle |
| <input-toggle></input-button> | Toggle with label |
| <input-toggle></input-button> | Toggle with `disabled` attribute |
